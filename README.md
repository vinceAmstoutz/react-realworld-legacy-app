Purpose
-------
:whale: Docker Workshop to dockerize a legacy app with a simple [Dockerfile](Dockerfile).

Getting started
---------------
1. `docker build -t myapp . `
2. `docker container run -p 4100:4100 --name myapp myapp`

:tada: The app is now working on http://localhost/4100 !
